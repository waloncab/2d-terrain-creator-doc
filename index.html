Create

In order to create your first terrain you need to setp up the tool. Go to S-Lines in the menubar and select S-Lines/Load, then wait a few seconds for the project to configure.

Now, if you reaload de S-Lines menubar you can see the the spline available. Create the 2D spline and start desinging the shape you want The tool is design with the spline as the core component. This means that for you to create the mesh or add prefabs, these must be extenden from the spline.

Spline

The spline is the core element of the tool. For better user expirence there are some key features that ensure its proper behaviour in a 2.5D enviroment. Only Forward movement available to ensure the mesh generates properly.

There are different configurations for the nodes placment so you can achive any desired shape. The following list shows some of them.

Node Behaviour
Free Leaf nodes can be drag anywhere
Aligned Leaf nodes are enforceb by their tangents
Mirrored Leaf nodes are enforced by tangents and same distance to center
Poly Leaf nodes disappear to make polygonal lines

As said before, in order to draw the mesh or to add prefabs to the terrain, these components must be extended from the spline. Select the requiered one in the extension menu and then click on "extend". By defautl each component is added to the scene as a root element, you can readjust in later to fit the hirarchy configuration that best fits you.

Extension Behaviour
Traverse Traverse the spline and keep GameObjects at a fixed state in the spline
Mesh Render different meshes along the spline

In order to see the full behaviour go to the Spline Tool Documentation

Mesh

Since the mesh is the most important feature in this tool, there are quite a bunch of things that you can modify. The Line_Mesh gather the common behaviour of the mesh, which is generated along the line, in this case the spline.

Check the full documentation for the Line Mesh component here Then use the extension behaviour to generate the different terrain shapes.

Extend

The way you select a concrete shape is by extending the Line Mesh component. Once selected click on "Extend", another script is added with concrete parameters for the selected mesh.

  1. Base
  2. Surface

Base

It creates a shape that goes below the spline into the the floor (0,0,0). It is the one you're porbabily going to use the most since it conforms the ground base.

Parameter Description
Width The width of the rect that conforms the base

Surface

In case you want you to use it, you can select from diferent typs of styles to customize the surface of the terrain.

Parameter Description
Width The width of the rect that conforms the surface
Sytle Description
Rocky To achive a random rock cut style
Wave Select the last node
Random Reset the spline

Box

A simple square shape that may be useful for creating structures such as bridges.

Prefab

To keep it as simple as posible (and as customizable as needed) the way to create prefabs is by using a traversal component and attach the gameObject as a child.

Check the full documentation for Traversal component here. The psoibilities are endless, attach your prefab gameObjects to the Traversal and create the result that is needed. Use options such as flipping the direction or the normal.

This way you can create any offset needed with the unity transform tool. Also any complex hirarchy can be achived. There is no cool interface, but is the only way I can assure there are no limitations for the user when it comes to create a complex prefab. Exmp: creating a coin that needs an offset to look hovering from the ground.
